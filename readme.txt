== Installation ==
Designfolio Installation process is like - you'll need a shortcode [jwt_designfolio] , Create a New Page and place the Shortcode. Designfolio will appear.

1. From WordPress Dashboard go to "Plugins>> Add New>> Uploads", select 'Designfolio.zip' file and upload it. Or,
2. Upload 'Designfolio.zip' to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4 To show all Products a shortcode [jwt_designfolio] needed, Create a New Page and place the Shortcode. Designfolio will appear.
